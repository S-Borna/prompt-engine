# Prompt Engine - Proposals

Denna mapp innehåller olika upplägg/förslag för prompt-motorn.

## Hur man lägger till ett nytt förslag

1. Skapa en ny mapp: `proposal-X/`
2. Lägg all kod och dokumentation där
3. Inkludera en `README.md` som beskriver upplägget

## Upplägg

Klistra in dina förslag från Claude Code här så organiserar vi dem.
